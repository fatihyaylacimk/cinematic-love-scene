# Cinematic Love Scene 🌙❤️

A cinematic romantic animation built using **HTML5 Canvas and JavaScript**.

The scene shows two stickman characters walking toward each other under a starry night sky. When they meet, heart particles rise into the sky creating a cinematic romantic moment.

## Features

• Stickman character animation  
• Starry night sky environment  
• Moon lighting effect  
• Heart particle system  
• Smooth Canvas animation  
• Cinematic visual atmosphere  

## Technologies

HTML5  
CSS3  
JavaScript  
Canvas API

## How to Run

1. Download the project
2. Open `index.html` in your browser

No installation required.
